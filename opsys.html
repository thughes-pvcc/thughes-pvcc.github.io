Operating Systems Terminology
TERM DEFINITION
Operating system The software that controls the overall operation of the computer including file management, the user interface, and the program execution environment.
Job Queue An ordered list of jobs waiting to be processed by a subsystem.
FIFO First In First Out, the simplest page replacement algorithm.
Interactive processing Allows the user to interact directly with the computer.
Time-sharing A method of allowing individuals at different terminals to access the same computer system simultaneously.
Multiprogramming When multiple programs execute at a time on a single device.
Application software Software designed to handle specific tasks for users.
System software Software designed to provide a platform for other software.
Utility software A software designed to help analyze, configure, optimize, or maintain a computer.
Graphical user interface A visual way of interacting with a computer and its systems through the use of symbols, visual metaphors, and pointing devices.
Shell A computer application software that the services of an operating system to an external user or another program.
Kernel A computer program at the core of a computers operating system and generally has complete control over everything in the system.
Device Driver A special kind of software program that controls a specific hardware device attached to a computer. It is essential for a computer to work properly
File manager A computer program that provides a user interface to manage files and folders.
Memory manager The process of controlling and coordinating a computers main memory. It ensures that blocks of memory space are properly managed and allocated so the operating system, applications, and other running processes have the memory they need.
Virtual memory A common technique used in a computers operating system. Virtual memory uses both hardware and software to enable a computer to compensate for physical memory shortages, temporarily transferring data from random access memory to disk storage.
Boot strapping To boot or to load a program into a computer using a much smaller initial program to load in the desired program.
ROM Read-Only Memory, refers to computer memory chips containing permanent or semi-permanent data.
Deadlock A situation in which two computer programs sharing the same resource are effectively preventing each other from accessing the resource, resulting in both programs ceasing to function.
Spooling A specialized form of multi-programming for the purpose of copying data between different devices.
Super user (or administrator) A special user account used for system administration.
Multi-factor authentication An electronic authentication method in which a user is granted access to a website or application only after successfully presenting two or more pieces of evidence to an authentication mechanism.
