All About GitHub

  • Q. What is a version control system?
    A. Software tools that help software teams manage changes to source code over time.
  • Q. What is Git?
    A. A free and open-source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.
  • Q. What is GitHub?
    A. A website and cloud-based service that helps developers store and manage their code, as well as track and control changes to their code.
  • Q. Who started GitHub and how was it started?
    A. Chris Wanstrath, PJ Hyett, Tom Preston-Werner, and Scott Chacon started GitHub and it was started out of necessity for their own work.
  • Q. What company owns it now?
    A. Microsoft
  • Q. How much does a GitHub account cost?
    A. For "Team" it is $48, for "Enterprise" it is $252
  • Q. What is the Octocat?
    A. The mascot for GitHub

Git/GitHub terms:

  • Repository- The most basic element of GitHub. Essentially a projects folder.
  • Commit- An individual change to a file (or set of files).
  • Fork- A personal copy of another users repository that lives on your account.
  • Push- This means to send your committed changes to a remote repository on GitHub.com.
  • Pull requests- Proposed changes to a repository submitted by a user and accepted or rejected by the repositorys collaborators.
  • Workflows- A configurable automated process that will run one or more jobs.
  • Issues- Suggested improvements, tasks, or questions related to the repository.
  • Raw button- You can view or copy the raw content of a file without any styling.
  • Blame button- Used to examine the contents of a file line by line and see when each line was last modified and who the author of the modifications was.
